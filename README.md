## CHAT-BPH 📌
[👮] **Autor**: Pedro Henrique.  
[📅] **Data**: 14/06/2026.  
[📌] **Tipo**: Informações CEP.  
[💻] **Deploy**: [LocalizaCEP](https://eupedrobarbosa03.github.io/localiza-cep/).

---

O LOCALIZACEP é uma aplicação web gratuita e segura para ver informações de um CEP. É possível ver Estado, localidade, UF, DDD, região, logradouro, bairro e o IBGE.  

---
### TECNOLOGIAS E CONCEITOS UTILIZADOS 💻
**[⚙️] Tecnologias**: HTML5, CSS3 e typescript.  
**[📗] Conceitos**: interafce-ts, API, poo, dom, regExp e modules.  
**[📁] Padrão**: POO e funções.

### FUNCIONALIDADES DO CHAT ✅
**[✅] Interface** agradável para o usuário.  
**[✅] Text animation** para o título de apresentação da aplicação. Efeito de digitação de texto.  
**[✅] Scroll animation** para efeito de carramento suave ao rolar a página.  
**[✅] Menu mobile** para dispositivos/telas menores.  
**[✅] Campo de preenchimento de CEP** para retorno de dados com validação.  
**[✅] Todas** as badeiras de todos os Estados do Brasil.    
**[✅] Cada bandeira** pisca ao retorno de um CEP que condiz com a UF.
